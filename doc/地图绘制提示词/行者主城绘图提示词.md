# 行者主城绘图提示词

## 使用说明

AI 很难一次性画出完整主城时，可以把主城拆成多个区域分别生成。每张图只画一个功能区，并保留道路接口，最后再按地图结构拼接成完整主城。

建议生成顺序：

1. 中央裂隙广场
2. 裂隙观测所
3. 工坊与装备区
4. 兽栏与契灵训练区
5. 技能馆与训练庭院
6. 档案馆
7. 远征大厅与小队集结区
8. 外城墙与防御入口

## 统一风格提示词

每个区域提示词前都可以加上以下统一风格描述：

```text
像素风 2D RPG 主城地图，俯视 3/4 视角，精致像素艺术，暗色灾后世界观，旧城市废墟与魔导机械设施拼接而成，石砖、钢铁、管线、符文灯、临时脚手架、蓝白色裂隙光与暖色灯火并存，末日前线据点但仍有秩序感，地图结构清晰，可用于游戏场景，不要文字标牌，不要 UI，不要人物特写，不要现代高楼，不要写实照片风。
```

## 1. 中央裂隙广场

```text
行者主城中央广场区域，画面中心是一座稳定的巨大裂隙锚点，蓝白色裂隙核心悬浮在环形石台上，周围有符文地面、机械固定环、能量导管和守护柱。广场四周连接多条石砖道路，边缘留出通往工坊区、观测所、远征大厅和居住区的道路接口。整体是主城核心地标，庄严、危险、神秘。
```

## 2. 裂隙观测所

```text
行者主城的裂隙观测所区域，一座半废墟半机械化的研究建筑，屋顶有星图仪、旋转观测环、悬浮档案碎片和蓝白色投影装置。建筑周围堆放异界残片、记录水晶、缆线和临时支架。场景突出研究、观测、选择异世界入口的功能，边缘保留通往中央裂隙广场的道路。
```

## 3. 工坊与装备区

```text
行者主城的工坊区，画面包含锻造炉、机械工作台、武器架、装甲零件、熔炉烟囱、齿轮设备、符文强化台和材料仓库。整体是废墟中重建出的装备制造区，暖色炉火与冷色裂隙能量形成对比。道路宽一些，方便玩家停留和交互，边缘保留通往中央广场和交易区的接口。
```

## 4. 兽栏与契灵训练区

```text
行者主城的兽栏和契灵训练区，由石墙围栏、木质棚屋、魔导笼舍、训练场、喂养槽和小型生态隔离区组成。不要画具体复杂宠物，只表现不同大小的空栏位、爪印、发光饲料箱、契约符文和训练器具。氛围温暖但带有异界感，边缘保留通往主城道路的接口。
```

## 5. 技能馆与训练庭院

```text
行者主城的技能馆区域，一座由旧学院废墟改造的训练设施，包含练武场、木桩、法术靶、符文圆阵、武器练习架和半开放讲堂。地面有被反复使用的战斗痕迹，墙边摆放技能卷轴、训练石碑和能量灯。整体感觉像行者固化技能灵感、练习战斗招式的地方。
```

## 6. 档案馆

```text
行者主城的档案馆区域，一座安静但神秘的建筑，内部外露出高大的书架、封存柜、漂浮的世界档案碎片、破损地图、记忆水晶和蓝白色记录光带。建筑部分嵌入旧城市废墟，像是临时修复的研究档案库。氛围偏冷静、神秘、知识感，边缘连接观测所和中央道路。
```

## 7. 远征大厅与小队集结区

```text
行者主城的远征大厅区域，宽阔的集结平台，包含任务公告板但不要显示文字、装备整备台、补给箱、撤离信标、队伍等待区和通往城外的传送门框架。这里是玩家组队、接取行动、准备进入异世界的地方。整体要有前线军事据点感，但仍保持像素 RPG 主城的可读性。
```

## 8. 外城墙与防御入口

```text
行者主城的外城墙和防御入口区域，破损旧城墙被钢铁支架和魔导护盾加固，城门外能看到被裂隙污染的荒地。城墙上有瞭望塔、符文炮台、防御灯、补丁式装甲板和应急路障。画面重点是主城边界、防御压力和末日前线感，保留一条通往城内中央道路的入口。
```

## 通用负面提示词

```text
完整大地图，过多区域塞在同一张图里，现代城市，高楼大厦，飞船，纯科幻空间站，过多人物，人物脸部特写，文字，UI，模糊，低清晰度，写实摄影风，过度卡通，空旷无细节。
```

## 拼接建议

先用“中央裂隙广场”确定整体色调、道路宽度和建筑比例。后续区域都围绕中央广场生成，提示词中明确保留道路接口。最终拼接时，可以让中央裂隙广场位于地图中心，观测所与档案馆靠近上方或右上方，工坊靠左侧，兽栏靠下方或外侧，远征大厅靠近城门，外城墙作为最外围边界。
